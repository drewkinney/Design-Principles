# Reflective / Arc — Loyalty / Return

Reflective arc is what the experience becomes in memory.
Completed flows generate loyalty. Broken flows generate avoidance.
The difference between a user who returns and one who doesn't is whether
the arc they remember felt worth completing again.

Score: PASSES | FAILS | MIXED | UNSCORED

---

## Evaluate

- Does the experience deliver on the emotional promise established at Visceral?
- Does the post-conversion state — confirmation, onboarding, follow-up — close the arc?
- Does the experience generate positive identity association: "this is mine"?
- Are return paths — login, reorder, continue — low-friction enough to feel like coming home?
- Would a user who completed the primary task describe the experience to someone else?
- Does brand identity remain legible throughout the full arc, or dissolve during utility sections?

Failure modes: confirmation screens that feel transactional rather than celebratory,
return user flows with no shortcuts or recognition, onboarding that re-establishes
what the user already knows, post-conversion states that abandon the emotional arc,
brand identity that evaporates during cart or account sections.

PASSES: arc closes completely; post-conversion delivers; return path is low-friction.
MIXED: arc closes but post-conversion or return path is functional rather than rewarding.
FAILS: arc is incomplete; no identity attachment formed; users have no reason to return.
