# Visceral Domain — First Contact → Trust

Visceral = first-contact impact. Trust and attraction established before any interaction occurs.
If Visceral fails, Behavioral starts in deficit. Reflective never recovers from a failed Visceral.

---

## Execution

Spawn in parallel. Collect 5 finding objects.

| Meta-Principle | File | Covers |
|---|---|---|
| Alignment | visceral/alignment.md | Aesthetic Effect |
| Signal | visceral/signal.md | Color & Psychology |
| Momentum | visceral/momentum.md | Common Fate |
| Order | visceral/order.md | Hierarchy / Sequence |
| Arc | visceral/arc.md | Experience / Emotion |

---

## Output

Return `visceral_findings[]` — five finding objects, one per principle.
Pass to HANDOFF.md for compilation with behavioral_findings[] and reflective_findings[].
